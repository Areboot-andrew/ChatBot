Системний Промпт

Головна інструкція для LLM. Визначає персону, тон та базові правила поведінки.

Параметри генерації (LLM)

Виберіть шаблон, щоб автоматично заповнити поля нижче.

0 = чіткі відповіді, 1 = креативні.

Залиште пустим для використання локального сервера.

Якщо API публічне і не вимагає ключа, впишіть будь-що (напр. 'free').

Бізнес-правила (Жорсткі обмеження)

Вкажіть правила, які бот ніколи не повинен порушувати (наприклад: "Ми не ремонтуємо принтери" або "Повернення тільки протягом 14 днів"). Вони будуть вшиті як жорсткі інструкції.

Маркетинг та Крос-сейл (Upsell)

Інструкції для допродажів. Наприклад: "Якщо клієнт купує відеокарту, ненав'язливо запитай, чи вистачить йому потужності блоку живлення". Бот застосує їх природно.

Політика Ескалації (Що робити, коли немає відповіді)

Як бот повинен поводитись, коли в базі знань або прайсі немає інформації.
