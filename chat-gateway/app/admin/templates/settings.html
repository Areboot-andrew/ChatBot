Системний Промпт

Головна інструкція для LLM. Визначає персону, тон та базові правила поведінки.

Параметри генерації (LLM)

Виберіть шаблон, щоб автоматично заповнити поля нижче.

0 = чіткі відповіді, 1 = креативні.

Залиште пустим для використання локального сервера.

Якщо API публічне і не вимагає ключа, впишіть будь-що (напр. 'free').

Бізнес-правила (Жорсткі обмеження)

Вкажіть правила саме цього бізнесу, які бот не повинен порушувати: обмеження асортименту, приймання замовлень, повернення, оплати чи інші зобов'язання.

Синоніми каталогу

Вирівнюють побутові слова клієнта до назв у вашому прайсі (щоб пошук знаходив). Рядок: слово=синонім,синонім. Порожньо = розумний дефолт.

Тон, інтейк і правила рішень — у головному промпті; пошук/відбір по кожному джерелу — у Логіці pipeline. Окремі старі поля для цього прибрані.

Реакція на грубість і бан сесії

Модуль вмикається галочкою. Бот сам визначає пряму образу/погрозу на адресу майстра; після вказаної кількості попереджень — бан, і двигун припиняє відповіді в цій сесії (бан з'являється в «Бани сесій»).

Надсилається один раз. На всі наступні повідомлення цієї сесії бот уже не відповідає.

Передача оператору (Ескалація)

Що бот каже/робить, коли спрацьовує роут handoff (клієнт просить людину). Порожньо = бот чесно скаже, що автоматичної передачі немає.

Архітектура розмови

Єдина схема роботи: чат-модель говорить з клієнтом, controller-модель вирішує чи потрібні дані, а кожен роут читає свою базу в окремій памʼяті й повертає очищений стан.

1. Чат-модель

Тон, мова, стиль, відповідь клієнту.

2. Controller

Чат + стан + карта знань. Вибір: відповідати чи відкрити роут.

3. Роут

Окрема памʼять, глибокий блок, фільтр фактів/умов.

4. Стан

Збережені факти, умови, попередження, бан сесії.

Пошук в інтернеті

Пошук іде через Google (за допомогою Serper.dev) — ключ уже вшитий, працює з коробки. Можете вписати свій ключ Serper (2500 запитів/міс безкоштовно), коли спільний вичерпається. Знайдені сторінки бот відкриває і читає повністю.

Це можуть бути постачальники, дистриб'ютори, оптові бази або маркетплейси. Дані цього джерела є сторонніми, доки промпт роута явно не визначить інше.

Якщо сайт має пошук через URL — впишіть шаблон із {query}. Бот підставить короткий запит, відкриє сторінку і прочитає доступні пропозиції. Загальний веб-пошук лишається запасним.

Як подавати зовнішню ціну, порядок сайтів і як відділяти зовнішню вартість від вашої ціни — опишіть у промпті відповідного роута в «Логіці pipeline».
