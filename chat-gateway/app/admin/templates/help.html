Як бот думає над повідомленням

Бот працює як агент: на кожне повідомлення він сам вирішує, де шукати відповідь, робить кілька кроків «на бекфоні», і тільки потім відповідає клієнту живою мовою. Етапи (видно в Тест-чаті):

  1. Роутер — модель повертає рішення: який інструмент викликати або що вже можна відповідати.
  2. Інструмент — виконується (каталог / база знань / інтернет / бізнес-факти).
  3. Крок повторюється, поки модель не збере достатньо фактів (ліміт кроків — у Налаштуваннях).
  4. Фінальна відповідь — модель пише клієнту, спираючись ТІЛЬКИ на зібрані факти.

Привітання («привіт», «дякую») інструментів не викликають — одразу відповідь у вашому тоні.

Інструменти агента

list_categoriesСписок категорій послуг (дешево, без цін) — щоб побачити, що ви взагалі робите.
search_catalogПошук по прайсу поетапно: за назвою послуги → за категорією → список категорій. Не вантажить усі ціни одразу.
search_knowledgeБаза знань: Q&A та документи (RAG).
web_researchПошук в інтернеті + реальний захід на знайдені сайти і читання тексту. Спершу по Довірених сайтах.
open_urlВідкрити конкретне посилання і прочитати його.
get_business_infoАдреса, години, телефон, оплата (з вкладки «Бізнес-факти»).
escalateПередати оператору.

Де що налаштовувати

Тон, персона, бізнес-правила, маркетинг, режим агента, інструменти, Serper-ключ⚙️ Налаштування Бота
Контакти, години, оплата (бот тягне на запит клієнта)📚 База знань → Бізнес-факти
Q&A, документи-мануали📚 База знань
Прайси (шаблон, експорт, імпорт XLSX/CSV з мапінгом)📋 Прайси та Послуги
Інтенти + інструкції міркування зі слотом (узагальнення)📚 База знань → Схема Логіки
Telegram-бот, Telegram особистий акаунт, чат на сайті📡 Канали

Приклади (як бот поведеться)

Клієнт: «привіт»

→ Без пошуку. Відповідь у вашому тоні: «Привіт. Що зламалось?»

Клієнт: «скільки коштує почистити ноутбук»

→ search_catalog(«чистка ноутбука») → ціна з прайсу + «точно після безкоштовної діагностики».

Клієнт: «ремонтуєте міксери?»

→ каталог → нема прямого → база знань → сайт/інтернет (Довірені сайти). Якщо підтвердилось, що робите дрібну побутову — «так, привозьте на діагностику», а не вгадане «ні».

Клієнт: «які у вас години роботи»

→ get_business_info(hours) → відповідь з ваших Бізнес-фактів (або із сайту texno.plus, якщо поле порожнє).
